# A-school-marks-management-system
 A C++ console application that stores student records and automatically calculates grades and percentages using array-based data handling.
Key points:

Built in C++ using arrays to store and manage student data (names, roll numbers, subject marks)
Calculates total marks, percentage, and assigns letter grades based on defined grading criteria
Includes functions for adding, displaying, and updating student records
Implements basic input validation to handle incorrect or out-of-range marks
Console/command-line-based interface for entering and viewing results
Demonstrates core programming concepts: arrays, functions, loops, and conditional logic
Technologies used: Basic C++ with array
